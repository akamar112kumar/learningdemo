what is decorator?

Decorators in Angular are a design pattern used to modify or annotate classes, properties, methods, and parameters with meta-data, without changing their behavior. Essentially, decorators allow developers to modify or extend the functionality of a class or its members through declarative syntax. They can be considered as wrappers around classes or class members.

Angular uses four main types of decorators:

Class decorators: Class decorators apply to classes to modify their behavior or define metadata. example

Property Decorators: Property decorators are used on class properties. They help configure properties and provide metadata for property bindings. example

Method decorators: Method decorators apply to class methods and are often used to react to lifecycle events. example

Parameter decorators: Parameter decorators apply to method or constructor parameters, providing additional metadata about the parameter example
